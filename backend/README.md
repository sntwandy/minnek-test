## Run Locally

Clone the project

```bash
  git clone [paste the https or ssh url]
```
Go to the task3 branch

```bash
  git checkout task3
```

Go to the project directory

```bash
  cd backend
```

Install packages

```bash
  pnpm install
```

Run the project

```bash
  pnpm dev
```
